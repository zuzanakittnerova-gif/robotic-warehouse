# BrightPick Fleet Simulator

## O projekte

BrightPick Fleet Simulator je aplikácia vytvorená vo Flutteri, ktorá simuluje koordináciu viacerých autonómnych skladových robotov.

Cieľom projektu bolo navrhnúť jednoduchý systém schopný:

* prideľovať balíky robotom,
* plánovať trasy,
* spravovať stav batérie,
* koordinovať pohyb viacerých robotov,
* monitorovať výkon flotily.

Projekt vznikol ako praktická ukážka prepojenia robotiky, softvéru a AI-asistovaného vývoja.

---

## Hlavné funkcionality

### Robot A a Robot B

* autonómny pohyb po sklade,
* ručné ovládanie,
* prideľovanie úloh,
* vyzdvihovanie a doručovanie balíkov.

### Fleet Management

* koordinácia viacerých robotov,
* rozdeľovanie balíkov podľa vzdialenosti,
* sledovanie stavu flotily.

### Správa batérie

* sledovanie kapacity batérie,
* automatický presun k nabíjacej stanici,
* automatické dobíjanie.

### Fleet Dashboard

* počet doručení,
* počet krokov,
* stav batérie jednotlivých robotov,
* počet nabíjaní,
* aktuálne úlohy robotov.

## AI prvky projektu

Počas vývoja boli využité AI nástroje na:

* návrh architektúry,
* ladenie a refaktorizáciu kódu,
* generovanie alternatívnych riešení,
* iteratívne vylepšovanie projektu.

AI nebola použitá ako náhrada vývojára, ale ako nástroj na zrýchlenie vývoja a testovania nápadov.
